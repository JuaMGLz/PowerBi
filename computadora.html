Introducción

Power BI Desktop es una herramienta de análisis y visualización de datos desarrollada por Microsoft, que permite transformar grandes volúmenes de información en gráficas, reportes interactivos y paneles dinámicos.

Su objetivo principal es ayudar a las empresas y usuarios a tomar decisiones basadas en datos, presentando la información de manera clara, visual y comprensible.

Con Power BI Desktop, puedes conectar diferentes fuentes de datos (como Excel, bases de datos SQL, servicios web o archivos CSV), limpiar y transformar la información, y luego crear visualizaciones que facilitan el análisis de tendencias, comparaciones y resultados.

Principales Ventajas:

🔗

Conexión con múltiples orígenes de datos (locales y en la nube).

⚙️

Transformación y modelado de datos mediante el lenguaje Power Query.

📊

Creación de dashboards personalizados con gráficos, mapas, indicadores y más.

📤

Integración con Power BI Service, lo que permite publicar y compartir informes en línea.

Evidencias de Power BI Desktop

A continuación se presentan las capturas de pantalla y evidencias del uso de Power BI Desktop para la creación de informes y análisis de datos.

Pantalla de inicio

Al abrir Power BI Desktop, se muestra la pantalla de inicio, que es el punto de partida para crear o abrir proyectos de análisis de datos. Esta interfaz está diseñada para que el usuario pueda acceder rápidamente a sus informes, conectarse a fuentes de datos o comenzar desde cero.

Funcionalidades principales: Crear informes en blanco, conectar a diferentes orígenes de datos (Excel, SQL Server, OneLake), acceder a informes recientes y contenido compartido.

Menú lateral izquierdo

El menú lateral izquierdo de Power BI Desktop proporciona acceso rápido a las funciones principales de navegación y configuración de la aplicación.

📁
Inicio:

Te lleva a la página principal (la que se muestra en la imagen).

📂
Abrir:

Permite acceder a archivos .pbix que ya hayas creado.

⚙️
Opciones inferiores:

Cerrar sesión, Configuración y Acerca de Power BI Desktop.

Panel de Visualizaciones

El panel de Visualizaciones en Power BI Desktop permite crear, personalizar y combinar gráficos (barras, columnas, líneas, mapas, etc.) y ajustar colores, etiquetas, leyendas y formatos para convertir datos en información clara y accionable.

3.1
Gráfico de Barras Apiladas

Compara categorías mostrando la composición de cada una dentro de un total. Ideal para visualizar proporciones y diferencias de forma compacta.

3.2
Gráfico de Columnas Apiladas

Muestra la contribución de cada valor en vertical, permitiendo comparar totales y proporciones por categoría con claridad.

3.3
Gráfico de Barras Agrupadas

Compara categorías lado a lado para ver diferencias entre grupos rápidamente; útil cuando el foco es la comparación directa de valores.

3.4
Gráfico de Barras 100% Apiladas

Enfatiza las proporciones dentro de un total normalizado (100%). Perfecto para comparar porcentajes entre categorías sin sesgo por tamaño.

3.6
Gráfico de Líneas

Muestra la evolución de los datos en el tiempo mediante líneas conectadas. Es ideal para analizar tendencias y cambios progresivos entre diferentes series.

3.7
Gráfico de Áreas

Similar al gráfico de líneas, pero resalta el área bajo la curva para enfatizar el volumen total representado. Es útil para destacar la magnitud acumulada de los datos.

3.8
Gráfico de Áreas Apiladas

Superpone áreas por categoría para mostrar la contribución de cada una al total acumulado. Ideal para observar cómo se componen las tendencias globales.

3.9
Gráfico de Áreas 100% Apiladas

Muestra la proporción de cada serie en relación con el total normalizado al 100%, permitiendo comparar la composición relativa entre categorías.

3.10
Gráfico de Columnas Apiladas y de Líneas

Combina columnas apiladas con una línea para mostrar la composición total junto con una tendencia general, facilitando el análisis de ambas métricas en un solo gráfico.

3.11
Gráfico de Columnas Agrupadas y de Líneas

Permite comparar categorías mediante columnas agrupadas, mientras que la línea representa una métrica de referencia o tendencia complementaria.

3.12
Gráfico de Cintas

Muestra cómo cambian las posiciones o rangos de las categorías a lo largo del tiempo mediante cintas visuales, resaltando subidas y bajadas en el orden.

3.13
Gráfico de Cascada

Representa incrementos y decrementos sucesivos que explican la variación entre un valor inicial y final. Es ideal para análisis financieros y de resultados acumulativos.

3.14
Gráfico de Embudo

Representa etapas secuenciales de un proceso (como ventas o conversiones), mostrando cómo disminuyen los valores a lo largo de cada fase. Ideal para análisis de flujos o embudos comerciales.

3.15
Gráfico de Dispersión

Muestra la relación entre dos o más variables mediante puntos distribuidos en un plano. Permite identificar correlaciones, patrones o grupos dentro de los datos.

3.16
Gráfico Circular

Divide un total en porciones proporcionales, facilitando la comparación de partes respecto al todo. Ideal para visualizar porcentajes de participación o distribución.

3.17
Gráfico de Anillos

Similar al gráfico circular, pero con un espacio central vacío. Permite incluir etiquetas o totales en el centro y mejora la legibilidad de los valores representados.

3.18
Treemap

Muestra jerarquías y proporciones con rectángulos anidados; el tamaño y color representan peso y categoría.

3.19
Mapa

Ubica puntos por coordenadas o nombres geográficos; útil para analizar distribución espacial de métricas.

3.20
Mapa Coroplético

Colorea regiones (país, estado, municipio) según valor; ideal para comparar intensidades por zona.

3.21
Azure Maps

Visualización cartográfica avanzada con capas, estilos y capacidades geoespaciales integradas en la nube de Microsoft.

3.22
Medidor

Representa el progreso de un valor frente a una meta establecida. Ideal para visualizar el cumplimiento de objetivos o indicadores de rendimiento.

3.23
Tarjeta

Muestra un solo valor destacado, como una suma, promedio o porcentaje. Es útil para resaltar métricas clave de forma rápida y visible.

3.24
Tarjeta de Varias Filas

Permite mostrar múltiples métricas o valores clave dentro de un mismo recuadro, facilitando la comparación de datos en un formato compacto.

3.25
KPI

Los indicadores clave de rendimiento (KPI) permiten evaluar el desempeño frente a una meta o tendencia, mostrando el estado mediante íconos y colores visuales.

3.26
Segmentación de Datos

Filtro visual interactivo para acotar la vista del reporte por categorías, fechas u otros campos. Afecta a los visuales conectados en la página.

3.27
Tabla

Presenta datos en filas y columnas con ordenación, formato y totales. Útil para listas detalladas y ver valores exactos.

3.28
Matriz

Tabla dinámica con niveles (filas/columnas jerárquicas) y totales parciales. Ideal para análisis tipo “pivot”.

3.29
Objeto Visual de Script de R

Permite crear visualizaciones personalizadas usando código R. Requiere tener R instalado y configurar los paquetes necesarios.

3.30
Objeto Visual de Python

Permite generar visualizaciones personalizadas mediante scripts de Python. Es útil para análisis avanzados y modelos predictivos dentro de Power BI.

3.31
Elementos Influyentes Clave

Identifica automáticamente los factores que más impactan en una métrica o resultado. Utiliza inteligencia artificial para encontrar patrones y relaciones en los datos.

3.32
Esquema Jerárquico

Desglosa una métrica principal en diferentes niveles o categorías, facilitando la exploración de causas y la detección de jerarquías dentro de los datos.

3.33
Preguntas y Respuestas

Permite hacer consultas en lenguaje natural sobre los datos. Power BI genera automáticamente una visualización en función de la pregunta escrita por el usuario.

3.34
Narrativa

Genera descripciones automáticas de los datos mediante inteligencia artificial. Resume tendencias y resultados clave en texto dentro del informe.

3.35
Goals (Versión Preliminar)

Permite crear y monitorear objetivos empresariales dentro del panel de Power BI. Conecta métricas y seguimiento visual del progreso hacia metas establecidas.

3.36
Informe Paginado

Diseñado para informes largos y detallados que abarcan varias páginas, con control preciso de formato, encabezados, pies y exportación a PDF.

3.37
Tarjeta (Nueva)

Versión actualizada de la visualización de tarjeta, con opciones de diseño mejoradas, soporte para íconos dinámicos y mayor personalización visual.

3.38
Segmentación de Botones

Filtro tipo “toggle” con botones personalizables para cambiar vistas o categorías de forma rápida y visual.

3.39
Segmentación de Texto

Slicer para escribir y filtrar por coincidencias de texto (contiene, comienza con, etc.). Útil para búsquedas rápidas.

3.40
Segmentación de Lista

Filtro en forma de lista (simple o multi-selección) para acotar la información por valores discretos.

3.41
ArcGIS for Power BI

Integración cartográfica de Esri para análisis espacial avanzado: capas, geocodificación y mapas temáticos profesionales.

3.42
Power Apps para Power BI

Integra aplicaciones interactivas creadas con Power Apps directamente dentro de Power BI. Permite capturar, modificar o actualizar datos en tiempo real desde los informes.

3.43
Power Automate for Power BI

Conecta flujos automáticos de Power Automate con tus informes. Permite ejecutar acciones, notificaciones o procesos directamente desde las visualizaciones de Power BI.
